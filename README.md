# WordClock_Tutorial
Mit Hilfe des Tutorial soll auf einfachen Weg eine WordClock entwickelt werden. Das Tutorial ist speziell für Anfänger ausgelegt und beinhaltet eine Einführung in folgende Themen: Kommandozeile, Linux Filesysteme,  Python, MicroPython, ESP8266/ESP32 und einfache Mikroelektronik Bastelei.
